usage: jira_reader.py [-h] [--jql JQL] [--filter-id FILTER_ID]
                      [--view-id VIEW_ID] [--limit LIMIT] [--offset OFFSET]
                      [--no-cache] [--force] [--cached-only] [--output OUTPUT]
                      [--debug-log DEBUG_LOG]

Fetch Jira tickets from JQL, saved filter, and/or Polaris view

options:
  -h, --help            show this help message and exit
  --jql JQL             JQL query string
  --filter-id FILTER_ID
                        Jira saved filter ID (numeric)
  --view-id VIEW_ID     Numeric Polaris view ID
  --limit LIMIT         Max tickets per source (default: 200)
  --offset OFFSET       Pagination offset
  --no-cache            Bypass JQL cache for view resolution
  --force               Skip timestamp checks, fetch and re-summarize
                        everything
  --cached-only         Output cached summaries from DB without API fetching
                        (fast, for reports)
  --output OUTPUT       Write results to this file (default: data/jira-
                        output.md)
  --debug-log DEBUG_LOG
                        Write debug messages to this file (default: data/jira-
                        debug.log)
